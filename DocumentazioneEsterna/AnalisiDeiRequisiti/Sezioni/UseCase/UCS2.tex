
\subsubsection{UCS 2 - Logout dal server}

\begin{itemize}
\item \textbf{Attori primari:} Amministratore \glo{autenticato}
\item \textbf{Precondizione:} L'amministratore è \glo{autenticato} presso il server.
\item \textbf{Postcondizione:} L'amministratore non è più \glo{autenticato} presso il sistema.
\item \textbf{Scenario principale:} L'amministratore seleziona la funzionalità di \glo{logout} per disconnettersi dal server.
\end{itemize}
