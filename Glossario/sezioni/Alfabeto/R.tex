\section{R}
\TermineGlossario{Rancher}
\DefinizioneGlossario{È una piattaforma di gestione per container Docker. Include una distribuzione Kubernetes e l'opzione tra Docker Swarm e Apache Mesos. Tutti utilizzano Docker come runtime del contenitore sottostante e coordinano i contenitori in esecuzione tra più nodi fisici discreti. Rancher include anche servizi di infrastruttura modulare tra cui networking, bilanciamento del carico, rilevamento di servizi, monitoraggio e ripristino.}

\TermineGlossario{React}
\DefinizioneGlossario{È una libreria open source di JavaScript per creare interfacce utente (UI).}

\TermineGlossario{Redis}
\DefinizioneGlossario{È un archivio di strutture di dati in memoria open source, utilizzato come database, cache e broker di messaggi. Supporta strutture di dati come stringhe, hash, elenchi, set, 
set ordinati con query di intervallo, bitmap, hyperloglogs, indici geospaziali con query di raggio e flussi.}

\TermineGlossario{Refresh manuale}
\DefinizioneGlossario{L'utente di propria iniziativa invia una richiesta di aggiornamento della lista delle organizzazioni.}

\TermineGlossario{Regressione lineare (RL)}
\DefinizioneGlossario{La regressione formalizza e risolve il problema di una relazione funzionale tra variabili misurate sulla base di dati campionari estratti da un'ipotetica popolazione infinita.}

\TermineGlossario{Repository}
\DefinizioneGlossario{È un archivio dove vengono raccolti dati ed informazioni in formato digitale sulla base di metadati che ne permettono la rapida individuazione.}

\TermineGlossario{Representational State Transfer (REST)}
\DefinizioneGlossario{È uno stile architetturale software per i sistemi distribuiti e rappresenta un sistema di trasmissione di dati su HTTP senza ulteriori livelli; non prevede il concetto di sessione (stateless).}

\TermineGlossario{Requisiti}
\DefinizioneGlossario{È l'insieme di tutte le funzionalità che il prodotto software deve offrire e che sono definite nelle specifiche del proponente. I requisiti di possono dividere in tre categorie: obbligatori, desiderabili ed opzionali.}

\TermineGlossario{Responsabile}
\DefinizioneGlossario{Ha l'incarico di pianificare, motivare, coordinare e controllare i membri del team. Il suo compito prevede inoltre l'approvazione dei documenti e l'emanazione di piani e scadenze. Ha l'onere di rappresentante del team presso l'azienda proponente.}

\TermineGlossario{Rete neurale (artificiale)}
\DefinizioneGlossario{Nel campo del machine learning, una rete neurale artificiale è un modello computazionale composto di "neuroni" artificiali, ispirato vagamente dalla semplificazione di una rete neurale biologica.}
