\section{J}
\TermineGlossario{JavaScript}
\DefinizioneGlossario{È un linguaggio di scripting orientato agli oggetti e agli eventi, comunemente utilizzato nella programmazione Web lato client, per la creazione, in siti web e applicazioni web, di effetti dinamici interattivi tramite funzioni di script invocate da eventi innescati a loro volta in vari modi dall'utente sulla pagina web in uso.}
\TermineGlossario{JavaScript Object Notation (JSON)}
\DefinizioneGlossario{È un formato adatto all'interscambio di dati fra applicazioni client/server ed è basato sul linguaggio JavaScript, ma ne è indipendente.}
